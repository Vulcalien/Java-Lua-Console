Variables

scr_w, scr_h

scr_w - the screen width
scr_h - the screen height

font_w, font_h

font_w - the width of font's characters (the font is monospaced)
font_h - the height of font's characters

map_w, map_h

map_w - the width of the map
map_h - the height of the map


General Functions

loadscript(script)

  1. script (String) - the path of the script to be loaded
Calls 'dofile' for the given script.
The script path is relative to 'console-userdata/script'.

returns: nil

key(id)

  1. id (int) - the id of the key
key(0) -> W
key(1) -> A
key(2) -> S
key(3) -> D

returns: true if the key is pressed, else false

sfx(name)

  1. name (String) - the name of the sound. The '.wav' hasn't to be added
Plays a sound contained in the 'sfx' folder.
Prints an error message if the sound does not exist.

Example:
sfx("test")
will play 'test.wav'.

returns: nil

Map Functions

gettile(x, y)

  1. x (int) - tile's x
  2. y (int) - tile's y
returns: the tile's id (int)

settile(x, y, id)

  1. x (int) - tile's x
  2. y (int) - tile's y
  3. id (int) - tile's id
Sets to [id] the tile at [x, y]

returns: nil

maprender([scale [, x [, y)

  1. scale (int) def: 1 - the render scale
  2. x (int) def: 0 - the render offset x
  3. y (int) def: 0 - the render offset y
Renders the map.

returns: nil
