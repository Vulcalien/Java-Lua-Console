Lua Documentation

Variables

scr_w, scr_h

scr_w - the screen width
scr_h - the screen height

font_w, font_h

font_w - the width of font's characters (the font is monospaced)
font_h - the height of font's characters

map_w, map_h

map_w - the width of the map
map_h - the height of the map


General Functions

loadscript(script)

  1. script (String) - the path of the script to be loaded
Calls 'dofile' for the given script.
The script path is relative to 'console-userdata/script'.

returns: nil

key(id)

  1. id (int) - the id of the key
key(0) -> W
key(1) -> A
key(2) -> S
key(3) -> D

returns: true if the key is pressed, else false

sfx(name)

  1. name (String) - the name of the sound. The '.wav' hasn't to be added
Plays a sound contained in the 'sfx' folder.
Prints an error message if the sound does not exist.

Example:
sfx("test")
will play 'test.wav'.

returns: nil


Screen Functions

settransparent(color)

  1. color (int) - the color that will be ignored
Sets the color to be ignored.
It is useful if you use a background color for sprites.

returns: nil

clear(color)

  1. color (int) - the color that will be used to clear the screen
Clears the screen.

returns: nil

pix(x, y, color [, w [, h)

  1. x (int) - the x of the pixel
  2. y (int) - the y of the pixel
  3. color (int) - the color of the pixel
  4. w (int) def: 1 - the width of the pixel
  5. h (int) def: 1 - the height of the pixel
Sets a pixel in the screen.
It can also draw a rectangle if [w] and [h] are passed.

returns: nil

write(text, color, x, y)

  1. text (String) - the text to be printed
  2. color (int) - the foreground color
  3. x (int) - indicates the left side of the text
  4. y (int) - indicates the top side of the text
Prints a text on the screen.

returns: nil

spr(id, x, y [, scale [, sw [, sh)

  1. id (int) - the id of the sprite (a number from 0 to 255); the corresponding coordinates are x = id % 16 and y = id / 16
  2. x (int) - the x draw coordinate
  3. y (int) - the y draw coordinate
  4. scale (int) - the rendering scale of the sprite
  5. sw (int) def: 1 - the width of the sprite (multiple of 8; on atlas)
  6. sh (int) def: 1 - the height of the sprite (multiple of 8; on atlas)
Draws a sprite in the screen at [x, y] coordinates.
The id indicates the position on the atlas (which has 256 sprites, from 0 to 255)
sw and sh are indicated as multiple of 8.
For example, sw = 3 means that the sprite's width is 3 * 8 = 24 pixels on atlas.
Even if sw or sh are not 1, [x, y] indicates the top-left sprite.

returns: nil


Map Functions

gettile(x, y)

  1. x (int) - tile's x
  2. y (int) - tile's y
returns: the tile's id (int)

settile(x, y, id)

  1. x (int) - tile's x
  2. y (int) - tile's y
  3. id (int) - tile's id
Sets to [id] the tile at [x, y]

returns: nil

maprender([scale [, x [, y)

  1. scale (int) def: 1 - the render scale
  2. x (int) def: 0 - the render offset x
  3. y (int) def: 0 - the render offset y
Renders the map.

returns: nil
